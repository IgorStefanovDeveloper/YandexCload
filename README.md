How to install app:
1) composer update
2) npm install
3) npm run build
4) create folders ['upload', 'upload/tmp', 'upload/img', 'Modules/Clouddiskprovider/Tokens']
5) create files ['.yandexdisk_auth', '.yandexdisk_secret'] in folder ['Modules/Clouddiskprovider/Tokens'] with your api tokens
Complete!
